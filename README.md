# sales-performance-dashboard

# Overview
This project presents an interactive Sales Performance Dashboard built using Tableau to analyze business data and derive actionable insights.

# Objectives
Analyze overall sales and profit performance
Identify top-performing products and categories
Compare sales across different regions
Track sales trends over time

# KeyFeatures
🔹 KPI Metrics: Total Sales, Profit, and Quantity
🔹 Time-based Sales Trend Analysis
🔹 Region-wise Sales Distribution
🔹 Profit Analysis by Category
🔹 Top 10 Products by Sales
🔹 Sales vs Profit Relationship (Scatter Plot)
🔹 Interactive Filters for dynamic analysis

# Tools & Technologies
Tableau (Data Visualization)
CSV Dataset (Data Source

# Key Insights
Technology category contributes the highest profit
Certain categories show low or negative profit margins
Sales performance varies significantly across regions
A small number of products generate a large portion of total sales

# Conclusion
This dashboard enables quick and effective analysis of sales data, supporting better decision-making through clear and interactive visualizations.
